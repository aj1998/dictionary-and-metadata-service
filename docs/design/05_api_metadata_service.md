# 05 — Metadata Service API

FastAPI service exposing structured queries on Postgres metadata (authors, shastras, anuyogas, teekas, books, pravachans).

## Service identity

- **Module path**: `services/metadata_service/`
- **Default port**: `8001`
- **OpenAPI**: `GET /openapi.json` (auto-generated by FastAPI)
- **Health**: `GET /healthz` → `{"status": "ok"}`
- **Base path**: `/v1`
- **Auth**: none on `GET`. `POST/PATCH/DELETE` are mounted under `/v1/admin/...` and require basic auth (see `15_deployment.md`).

## Pydantic response models (shared)

```python
class Multilingual(BaseModel):
    lang: str    # ISO-639-3
    script: str  # ISO-15924
    text: str

class Pagination(BaseModel):
    total: int
    limit: int
    offset: int
```

All list endpoints accept `limit` (default 50, max 200) and `offset` (default 0).

## Endpoints

### Authors

```
GET  /v1/authors                    list authors
GET  /v1/authors/{id|natural_key}   fetch one
POST /v1/admin/authors              create
PATCH /v1/admin/authors/{id}        update
DELETE /v1/admin/authors/{id}       only if no shastras reference it
```

**Author response:**
```json
{
  "id": "uuid",
  "natural_key": "kundkundacharya",
  "display_name": [{"lang": "hin", "script": "Deva", "text": "कुन्दकुन्दाचार्य"}],
  "kind": "acharya",
  "bio": [{"lang": "hin", "script": "Deva", "text": "..."}],
  "created_at": "2026-05-01T12:00:00Z",
  "updated_at": "2026-05-01T12:00:00Z"
}
```

### Shastras

```
GET  /v1/shastras                                list (filter: ?author_id=, ?anuyoga=, ?q=)
GET  /v1/shastras/{id|natural_key}               fetch
GET  /v1/shastras/{id|natural_key}/teekas        list teekas of this shastra
GET  /v1/shastras/{id|natural_key}/gathas        list gathas (paginated)
POST /v1/admin/shastras                          create
PATCH /v1/admin/shastras/{id}                    update
```

**Shastra response (expanded):**

### Search (admin)

```
GET /v1/admin/search?q=...&types=shastra,author,teeka
```

Returns lightweight matches across metadata tables using `pg_trgm`. Used by the admin UI's global search box.

## Example handler

```python
# services/metadata_service/routers/shastras.py
@router.get("/{ident}", response_model=ShastraDetail)
async def get_shastra(ident: str, session: AsyncSession = Depends(get_session)):
    shastra = await shastras_svc.get_by_id_or_natural_key(session, ident)
    if shastra is None:
        raise NotFound("shastra", ident)
    return ShastraDetail.from_orm(shastra)
```
